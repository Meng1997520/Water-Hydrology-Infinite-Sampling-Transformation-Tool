# Water-Hydrology-Infinite-Sampling-Transformation-Tool

## Description
通过模特卡罗模拟来预测水文活动

## Features
- 支持在Windows上运行


## Installation
### Requirements
- **Operating System**: Windows 10 or later
- **.NET Framework**: .NET 4.8
- **Visual Studio**: 2019 or later (with WPF development workload)

##Notes
-模特卡罗模拟需要大量重复抽样和计算，需要较高的计算资源和时间。
